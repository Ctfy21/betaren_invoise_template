## Лидеры по устойчивости

### Комплексная устойчивость (к ржавчинам + вирусам)
1. **[[Ермоловка]]** — жёлтая, бурая, стеблевая ржавчина + вирусы (4 группы)
2. **[[Система]]** — жёлтая, стеблевая ржавчина + вирусы, толерантна к бурой (4 группы)
3. **[[Сократ]]** — жёлтая ржавчина + вирусы, толерантен к бурой

### Бурая ржавчина (наиболее распространённая)
1. **[[Володя]]** — высокая устойчивость
2. **[[Интеза]]** — высокая устойчивость
3. **[[Ермоловка]]** — высокая устойчивость

### Септориоз
1. **[[Изумруд Дубовицкого]]** — устойчив
2. **[[ДФ 2020]]** — устойчив

### Головня
1. **[[Изумруд Дубовицкого]]** — устойчив
2. **[[ДФ 2020]]** — устойчив

## Уязвимости, закрываемые технологией

Восприимчивость к отдельным болезням компенсируется [[Технологическая карта|технологической картой]]:
- **Бурая ржавчина** (у Изумруда, ДФ 2020) → [[Фунгициды]]: Беназол (фаза 24), Костандо + Арго Прим (фаза 31), Триада (фаза 39)
- **Мучнистая роса** → те же фунгицидные обработки
- **Септориоз** → [[Фунгициды]]: Триада (фаза 39), Эйс (фаза 53–57)

## Рекомендации для КП

- Клиент жалуется на **болезни** → [[Система]] или [[Ермоловка]] (комплексная устойчивость)
- В регионах с **бурой ржавчиной** → [[Володя]], [[Интеза]]
- В регионах с **клопом-черепашкой** → только [[ДФ 2020]]
