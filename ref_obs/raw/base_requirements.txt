Как работает приложение для автоматизации КП
Суть в одном предложении
Продажник за 2 минуты заполняет 5 полей — приложение само собирает КП, считает цифры и отправляет клиенту.

Шаг 1 — Ввод данных (продажник, ~2 мин)
Простая форма на экране, 5 обязательных полей:
Поле
Пример
Регион
Оренбургская область
Площадь
1 500 га
Сорт 
Главная боль
Засуха / болезни / качество
Контакт клиента
email + телефон
Никаких лишних кнопок. Вводишь — нажал «Сформировать КП».

Шаг 2 — Логика внутри (автоматически, ~секунды)
Приложение пробегает по базе знаний и делает 4 вещи одновременно:

2. Считает объёмы — площадь × норма высева = кг семян. Площадь × технологический пакет = объём СЗР и удобрений. Автоматически считает итоговую сумму.
3. Подбирает аргументы — по боли клиента вставляет нужный блок:
Засуха → данные по засухоустойчивости сорта + кейс из похожего региона
Болезни → устойчивость к конкретным патогенам + схема защиты
Качество → данные по протеину/клейковине + лабораторные результаты
4. Вставляет кейс (это на будущее)  — ближайший географически реальный результат (Оренбург 83 ц/га, Ермоловка 140 ц/га и т.д.)

Шаг 3 — Выход: готовое КП
КП формируется как PDF (или ссылка на красивую веб-страницу) и содержит:
1. Шапка: логотип Щёлково × azimaya.ru, имя клиента
2. Рекомендованный сорт
3. Технологическая карта: семена + СЗР + удобрения
4. Расчёт объёмов и итоговая стоимость пакета
5. Кейс из региона: «В вашем регионе аналогичный результат...»
6. Следующий шаг: контакт агронома + кнопка «Записаться на выезд»
7. Подпись: А.И. Прянишников / агроном ФИО
