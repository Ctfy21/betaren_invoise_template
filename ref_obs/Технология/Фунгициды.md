# Фунгициды

#технология #фунгициды #болезни

> Препараты для защиты от грибных заболеваний. В [[Технологическая карта|технологической карте]] — 4 фунгицидные обработки, покрывающие весь вегетационный период.

## Препараты

### Беназол
| Параметр | Значение |
|----------|----------|
| Норма | 0.6 л/га |
| Цена | 1 350 руб/л |
| Фаза | 24 (кущение, осень) |
| Обработка | [[Технологическая карта#Обработка 3\|№3]] |

Первая фунгицидная обработка. Профилактика осенних инфекций.

### Арго Прим
| Параметр | Значение |
|----------|----------|
| Норма | 0.5 л/га |
| Цена | 3 600 руб/л |
| Фаза | 31 (1-й узел, весна) |
| Обработка | [[Технологическая карта#Обработка 6\|№6]] |

Весенняя обработка. Совместно с [[Гербициды#Костандо|Костандо]], [[Биостимуляторы#Биостим Зерновой|Биостим Зерновой]], [[Инсектициды#Фаскорд|Фаскорд]].

### Триада
| Параметр | Значение |
|----------|----------|
| Норма | 0.6 л/га |
| Цена | 2 800 руб/л |
| Фаза | 39 (флаг-лист) |
| Обработка | [[Технологическая карта#Обработка 7\|№7]] |

Ключевая обработка на флаг-листе. Защита верхнего яруса перед колошением. Совместно с [[Инсектициды#Фаскорд|Фаскорд]].

### Эйс
| Параметр | Значение |
|----------|----------|
| Норма | 1.0 л/га |
| Цена | 3 200 руб/л |
| Фаза | 53–57 (колошение) |
| Обработка | [[Технологическая карта#Обработка 8\|№8]] |

Финальная фунгицидная обработка. Защита колоса. Совместно с [[Инсектициды#Эсперо|Эсперо]].

## Схема фунгицидной защиты по фазам

```
Осень         Весна                    Лето
  24            31          39          53-57
  │             │           │            │
  Беназол       Арго Прим   Триада       Эйс
  0.6 л/га      0.5 л/га    0.6 л/га     1.0 л/га
```

## Связь с устойчивостью сортов

Двойная весенне-летняя обработка (Арго Прим + Триада + Эйс) компенсирует восприимчивость:
- К **бурой ржавчине**: [[Изумруд Дубовицкого]], [[ДФ 2020]]
- К **мучнистой росе**: большинство сортов (средняя устойчивость)
- К **септориозу**: [[Ермоловка]] (восприимчив)

Для сортов с высокой устойчивостью ([[Система]], [[Ермоловка]] — ржавчины) возможна оптимизация схемы.
См. [[Устойчивость к болезням]]
