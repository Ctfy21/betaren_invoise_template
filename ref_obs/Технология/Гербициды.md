# Гербициды

#технология #гербициды #сорняки

> Препараты для борьбы с сорняками. В [[Технологическая карта|технологической карте]] — 3 гербицидные обработки (осень + 2 весны).

## Препараты

### Костандо
| Параметр | Значение |
|----------|----------|
| Норма | 0.2 л/га (осень) / 0.4 л/га (весна) |
| Цена | 5 800 руб/л |
| Фазы | 13–14 (осень), 31 (весна) |
| Обработки | [[Технологическая карта#Обработка 2\|№2]], [[Технологическая карта#Обработка 6\|№6]] |

Применяется дважды: осенью в сниженной норме (0.2) и весной в полной (0.4). Основной гербицид схемы.

### Азорро
| Параметр | Значение |
|----------|----------|
| Норма | 1.0 л/га |
| Цена | 2 800 руб/л |
| Фаза | 25 (кущение, весна) |
| Обработка | [[Технологическая карта#Обработка 4\|№4]] |

Весенняя гербицидная обработка. Совместно с [[Удобрения и микроудобрения#Ультрамаг Фосфор Супер|Ультрамаг Фосфор Супер]], [[Биостимуляторы#Гумат калия Суфлер|Гумат калия Суфлер]].

## Схема гербицидной защиты

```
Осень         Весна
  13-14         25           31
  │             │            │
  Костандо      Азорро       Костандо
  0.2 л/га      1.0 л/га     0.4 л/га
```

Три обработки обеспечивают контроль сорняков на протяжении всей вегетации.
