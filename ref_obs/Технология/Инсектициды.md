# Инсектициды

#технология #инсектициды #вредители

> Препараты для защиты от вредителей. В [[Технологическая карта|технологической карте]] — 3 инсектицидные обработки (весна × 2 + лето).

## Препараты

### Фаскорд
| Параметр | Значение |
|----------|----------|
| Норма | 0.15 л/га |
| Цена | 2 400 руб/л |
| Фазы | 31 (1-й узел), 39 (флаг-лист) |
| Обработки | [[Технологическая карта#Обработка 6\|№6]], [[Технологическая карта#Обработка 7\|№7]] |

Применяется дважды в весенний период. Совместно с [[Фунгициды|фунгицидами]] (Арго Прим, Триада).

### Эсперо
| Параметр | Значение |
|----------|----------|
| Норма | 0.1 л/га |
| Цена | 4 800 руб/л |
| Фаза | 53–57 (колошение) |
| Обработка | [[Технологическая карта#Обработка 8\|№8]] |

Финальная инсектицидная обработка при колошении. Совместно с [[Фунгициды#Эйс|Эйс]].

## Схема инсектицидной защиты

```
Весна                    Лето
  31          39          53-57
  │           │            │
  Фаскорд    Фаскорд      Эсперо
  0.15 л/га   0.15 л/га    0.1 л/га
```

## Связь с сортами

Для [[ДФ 2020]] — толерантность к клопу-черепашке по хлебопекарным свойствам. Инсектицидная защита дополняет генетическую устойчивость.
См. [[Устойчивость к болезням]]
